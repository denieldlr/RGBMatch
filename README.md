# RGBMatch

A Python desktop application that automates RGB color matching by reading and processing Excel datasets. This project was inspired by an internal Excel-based tool I developed during my time at Pacific Paint (Boysen) Philippines and was recreated using Python to improve usability and maintainability.

## Features

- Imports RGB color data from Excel
- Fast color matching and lookup
- Desktop GUI built with Tkinter
- Data processing with pandas
- User-friendly interface for technical applications

## Technologies

- Python
- Tkinter
- pandas
- openpyxl
